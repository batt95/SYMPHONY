Using the NMAKE Utility

  • Edit the file user.mak to reflect your environment. Only minor edits should be required. An explanation of what has to be set is contained in the comments in the makefile. This basically requires the same routines that one needs to walk through in SYMPHONY's makefile. See the related parts of 2.2.3.4 section of SYMPHONY above.

  • Once configuration is done, type 
      nmake /f user.mak
    The executable user.exe will be created under the Debug directory.
  • To test the executable, type 
     Debug\user.exe -F ..\..\sample.user
