Microsoft Windows

First, follow the instructions for compiling the sample application in Section 8.5 to ensure you have the proper settings. Once the stub files in the SYMPHONY-3.0User hierarchy are filled in, use the project ``vrp'' as a template to create a new project called ``user'' that compiles the files in SYMPHONY-3.0User and creates a library. Modify the project ``symphony'' to compile and link with the new user library.
