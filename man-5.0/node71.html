Microsoft Windows

First, download SYMPHONY-5.0.zip and unzip the archive. This will create a subdirectory called SYMPHONY-5.0 containing all the source files together with the USER subdirectory. There are two options to get the executable. You can either compile on the command-line, using the MSVC++ makefile, USER WIN32 user.mak, or you can use the provided projects and workspaces. However for the second option, it is important the USER archive be kept in the SYMPHONY-5.0 subdirectory or the project files will not work. Compiling on the command-line is somewhat easier since it requires only editing the makefile and typing a single command.
