Using Purify and Quantify

The makefile is already set up for compiling applications using purify and quantify. Simply set the paths to the executables and type ``make pall'' or ``p*'' where * is the module you want to purify. The executable name is the same as described in Section 6.7.1, but with a ``p'' in front of it. To tell PVM to launch the purified version of the executables, you must set the parameters *_exe in the parameter file to the purified executable names. See Section 7.4.4 for information on setting parameters in the parameter file.
