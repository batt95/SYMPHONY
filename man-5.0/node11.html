API Overview

SYMPHONY 5.0 is the first version of SYMPHONY to be implemented as a callable library with a new interface derived from the COIN-OR Open Solver Interface. This change markedly improves SYMPHONY's usability and flexibility. Below, we briefly describe the new API, the C++ interface, and the use of the user callback functions.
