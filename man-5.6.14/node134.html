sym_set_defaults

[f]sym_set_defaults 
int sym_set_defaults(sym_environment *env)

Description:

This routine sets all the environment variables and parameters to their default values.

Arguments:

sym_environment *env INOUT Pointer to the SYMPHONY environment to be modified.

Return values:

FUNCTION_TERMINATED_ABNORMALLY Function invoked unsuccessfully.
FUNCTION_TERMINATED_NORMALLY Function invoked successfully.
