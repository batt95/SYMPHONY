Master Process parameters

M_verbosity - integer (0).

M_random_seed - integer (17).
A random seed just for the Master Process.

tm_exe, dg_exe - strings (``tm'', ``dg'').
The name of the executable files of the TM and DG processes. Note that the executable names may have extensions that depend on the configuration of the modules so these defaults may not work!

tm_debug, dg_debug - boolean (both FALSE).
Whether the processes should be started under a debugger or not.

tm_machine - string (empty string).
On which processor of the virtual machine the TM should be run. Leaving the parameter an empty string means arbitrary selection.

do_draw_graph - boolean (FALSE).
Whether to start up a DG process or not.

do_branch_and_cut - boolean (TRUE).
Whether to run the branch and cut algorithm or not. (Set this to FALSE to run heuristics only.)
