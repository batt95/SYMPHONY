Preparing for Sample Compilation.

  • Download the file SYMPHONY-3.0.1.tgz.

  • Unpack the distribution with ``tar -xzf SYMPHONY-3.0.1.tgz''. This will create a subdirectory called SYMPHONY-3.0.1/ containing the distribution.

  • Edit the makefile (SYMPHONY-3.0.1/Makefile) to reflect your environment. Only minor edits should be required. An explanation of what has to be set is contained in the comments in the makefile.
