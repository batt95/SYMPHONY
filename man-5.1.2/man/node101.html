Microsoft Visual C++

First, download SYMPHONY-5.1.2 and unpack the archive if it is required. You now have three options. You can either compile on the command-line using the automated MSDEV build system or NMAKE utility or you can use the provided projects and workspaces. For all of the following options, first go to the SYMPHONY Applications USER Win32 v6 directory.
