Unix

First, download, configure, and compile SYMPHONY as described in Section 2.2.2.5. This will generate the required library and makefiles for each application. After this, typing make in the SYMPHONY/Applications/USER/ subdirectory should successfully build the executable. For more information, including the parallel configuration instructions, see the SYMPHONY/Applications/USER/INSTALL file.
