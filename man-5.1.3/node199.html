sym_set_warm_start

[f]sym_set_warm_start 
int sym_set_warm_start(sym_environment *env, warm_start_desc *ws)

Description:

This routine is used to load a warm start structure to the environment.

Arguments:

sym_environment *env INOUT Pointer to the SYMPHONY environment.
warm_start_desc *ws IN Pointer to the warm start structure to be loaded to the environment.

Return values:

FUNCTION_TERMINATED_NORMALLY Function invoked successfully.
FUNCTION_TERMINATED_ABNORMALLY Function invoked unsuccessfully.
