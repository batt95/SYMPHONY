sym_get_user_data

[f]sym_get_user_data 
int sym_get_user_data(sym_environment *env, void **user)

Description:
This routine is used to get the user's problem data structure from SYMPHONY environment.

Arguments:

sym_environment *env INOUT Pointer to the SYMPHONY environment.
void **user OUT Pointer to the user defined problem structure.

Return values:

ERROR__USER Error in the passed in user structure.
FUNCTION_TERMINATED_ABNORMALLY Function invoked unsuccessfully
FUNCTION_TERMINATED_NORMALLY Function invoked successfully.
